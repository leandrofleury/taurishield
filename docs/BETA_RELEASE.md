# Beta Release Notes

The beta branch is intended for controlled testing.

## Beta acceptance criteria

- All manifests validate successfully.
- All manifests produce no High/Critical findings.
- Generated Tauri configuration has `withGlobalTauri = false`.
- Generated capabilities contain only explicitly enabled permissions.
- SARIF output is generated in CI.
- SBOM generation is available.
- Release evidence folder is generated by `release-check`.

## Not yet production-ready

- Full binary signing is documented but not automated end-to-end.
- Generated apps still require local Tauri toolchain validation.
- No automatic update framework is enabled.
- No centralized policy registry yet.
