# Security Policy

## Security posture

TauriShield is security-first and should fail closed.

## Blocked by default

- Shell access
- Filesystem access
- Clipboard access
- Downloads
- Camera
- Microphone
- Geolocation
- Global Tauri API exposure
- Telemetry
- Auto update

## Required controls

- HTTPS source URL
- Explicit allowlist
- CSP mode
- Policy evaluation before build

## Reporting vulnerabilities

Do not open public issues for sensitive vulnerabilities. Use the project's private security advisory flow once published.
